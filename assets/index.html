HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • If coding CSS outside of the HTML code, remember to call the document back in the HMTL code document .

Git

  • In Terminal: git status: checks what branch we are currently on
  • In Terminal: git checkout -b branch-name: creates a new branch and switches to it.
  • In Terminal: git add -A: will stage your changes to Git.
  • In Terminal: git pull origin main: will pull the remote main branch into the local main branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Javascript strings are immutable. This means that once a string is created, it is not possible to modify it.
  • Javascript arrays start on 0.
  • To comment out blocks of code add a "/*", to the start, and a "*/" to the end of the code block.
  • Remember to call the JS coding at the end of your HTML code.
